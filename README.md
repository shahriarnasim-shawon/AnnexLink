# AnnexLink

A Peer-to-Peer Service Marketplace for University Students.

AnnexLink is a web platform designed to connect university students who want to offer or request services within their campus community. The platform allows students to collaborate, share skills, and complete tasks in a secure and organized environment.

---

## Project Overview

Many students rely on informal social media groups to find help with tasks such as tutoring, design work, programming assistance, or event support. These methods are often unstructured and unreliable.

AnnexLink provides a centralized system where students can post services, request help, communicate with others, and complete tasks through a trusted campus-based platform.

---

## Objectives

- Create a centralized marketplace for campus-based services.
- Enable students to offer services and earn using their skills.
- Improve collaboration and networking within the university.
- Provide a secure platform for service exchange among students.
- Simulate a real-world online service marketplace.

---

## Key Features

- University email-based student registration and login
- User profile with skills, ratings, and reviews
- Service posting system for offering services
- Service request system for asking help
- Service browsing with search and filters
- Detailed service pages
- Real-time chat between users
- Review and rating system
- Personal dashboard for managing services and requests
- Admin panel for managing users and services
- Email notification system
- Secure payment system for service transactions
- Transaction history tracking
- Responsive design for mobile and desktop

## How It Works

1. Students create an account using their university email.
2. Users can post services or request help from other students.
3. Students browse available services or requests.
4. Users communicate through the platform chat system.
5. After completing a service, users can leave ratings and reviews.
